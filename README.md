# Pizza-Sales-Analysis-Using-SQL


## Project Overview

This project analyzes pizza sales data using SQL to uncover valuable business insights related to revenue, customer ordering behavior, and product performance. The analysis was performed using multiple SQL techniques including JOINs, GROUP BY, aggregate functions, subqueries, and window functions.

## Objectives

* Analyze overall sales performance and revenue trends.
* Identify best-selling and highest-revenue pizza types.
* Understand customer ordering patterns.
* Examine category-wise and size-wise sales distribution.
* Generate actionable insights for business decision-making.

## Key Analyses Performed

* Total number of orders placed
* Total revenue generated
* Highest-priced pizza identification
* Most commonly ordered pizza size
* Top 5 most ordered pizza types
* Category-wise quantity distribution
* Hourly order distribution
* Average pizzas ordered per day
* Top 3 pizza types by revenue
* Percentage contribution of each category to total revenue
* Cumulative revenue analysis
* Top revenue-generating pizzas within each category

## SQL Concepts Used

* SELECT Statements
* Aggregate Functions (SUM, COUNT, AVG)
* GROUP BY & ORDER BY
* INNER JOINs
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions
* Data Aggregation & Business Reporting

## Outcome

The project demonstrates how SQL can be used to transform raw transactional data into meaningful business insights, helping organizations make data-driven decisions regarding product offerings, sales strategies, and customer preferences.
